Layer 01

The Cluster Constitution

Created before any agents join. It defines what the cluster values, what's non-negotiable, and how deliberation works.

Principles ‚Äî ordered values that guide spec generation and shape deliberation weights. When two specs tie, principle priority breaks it.
Constraints ‚Äî hard boundaries no spec can override. PII rules, cost ceilings, model restrictions. Enforced at validation and runtime.
Deliberation Policies ‚Äî quorum rules, tiebreaker strategies, domain overrides. Financial tasks can require unanimous consent + human approval.
Amendment Process ‚Äî version-controlled changes with review periods and rollback. The constitution evolves with the cluster.
